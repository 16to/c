# 回调函数
回调函数是指通过函数指针调用的函数。它允许将函数作为参数传递给另一个函数，从而实现灵活的功能扩展。

```c
#include <stdio.h>
#include <stdlib.h>

void populateArray(int *arr, int size, int (*func)()) {
    for (int i = 0; i < size; i++) {
        arr[i] = func(); // 通过回调函数生成数组元素
    }
}

int getRandomNumber() {
    return rand() % 100; // 生成0到99之间的随机数
}

int main() {
    int arr[10];
    populateArray(arr, 10, getRandomNumber); // 将getRandomNumber作为回调函数传递

    // 输出生成的数组
    for (int i = 0; i < 10; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}
```
