# 数据类型

## 基本数据类型
- 整型（int）
char 1字节
unsigned char 1字节
int 4字节
unsigned int 4字节
short 2字节
unsigned short 2字节
long 8字节,32位系统4字节
unsigned long 8字节,32位系统4字节

- 浮点型（float）
float 4字节，6位有效数
double 8字节，15位有效数
long double 16字节，19位有效数

- 字符型（char）
- 布尔型（bool）

## 枚举类型

## void类型
1. 函数的返回类型 (如void func())
2. 指针类型 (如void *)
3. 函数参数 (如int func(void))

## 派生数据类型
- 指针类型
- 数组类型
- 结构体类型

## 类型转换
1. 隐式类型转换 (自动类型转换)
   低精度类型转换为高精度类型
   char -> short -> int -> long -> float -> double
2. 显示类型转换 (强制类型转换)
   (type)value
   type(value)
