# 枚举
```c
enum DAYS {SUN=1, MON, TUE, WED, THU, FRI, SAT} day;
```
或者
```c
enum {SUN=1, MON, TUE, WED, THU, FRI, SAT};
enum DAYS day;
```
1. 枚举类型是一种用户自定义的数据类型，用于表示一组相关的常量值。
2. 枚举类型的每个成员都有一个对应的整数值，默认从0开始递增，也可以手动指定初始值。
3. 枚举类型的变量只能取枚举成员的值，不能取其他整数值。

## 枚举遍历
```c
#include <stdio.h> 
enum DAYS {SUN=1, MON, TUE, WED, THU, FRI, SAT};
int main() {
    for (int i = SUN; i <= SAT; i++) {
        printf("%d ", i);
    }
    return 0;
}
```
## 枚举与switch语句
```c
#include <stdio.h>
enum DAYS {SUN=1, MON, TUE, WED, THU, FRI, SAT};
int main() {
    enum DAYS day = WED;
    switch (day) {
        case SUN:
            printf("Sunday\n");
            break;
        case MON:
            printf("Monday\n");
            break;
        case TUE:
            printf("Tuesday\n");
            break;
        case WED:
            printf("Wednesday\n");
            break;
        case THU:
            printf("Thursday\n");
            break;
        case FRI:   
            printf("Friday\n");
            break;
        case SAT:
            printf("Saturday\n");
            break;
        default:
            printf("Invalid day\n");
    }
    return 0;
}
